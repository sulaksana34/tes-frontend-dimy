### Menjalankan App

1. Install dulu package nya
   ```
   yarn
   ```
2. Jalankan pada mode development

### Perintah lainnya

  ```
  yarn build
  ```
- Lint Fix, untuk mempercantik kode, dan deteksi kesalahan kode
  ```
  yarn lint
  ```
